How to Use the Animation Tool

  1. Save/Load Project: Use the 'Save' and 'Load' buttons to save your progress as a .anim file and open it again later.
  2. Draw & Erase: Use the Pencil and Eraser tools on the left. Adjust color and brush size below them.
  3. Lasso Erase: Use the Lasso tool to draw a selection. When you release the mouse, everything inside the selection will be erased.
  4. Background: Use the 'Upload Image' button to add a picture background. Use 'Clear Background' to remove it.
  5. Add & Duplicate Frames: Use the '+' button in the timeline to add a new frame. Hover over a frame to find the copy icon to duplicate it.
  6. Onion Skin: Check this box in the toolbar to see faint outlines of your previous and *next* drawings, which helps with smooth animation.
  7. Play & Adjust FPS: Use the play button and FPS slider in the toolbar to preview your animation.
  8. Export GIF: When you're finished, use the 'Export GIF' button to save your animation as a downloadable .gif file.
