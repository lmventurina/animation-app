How to Use the Animation Tool

1. **Save/Load Project:** Use the 'Save' and 'Load' buttons to save your progress as a `.anim` file and open it later.

2. **Draw** on the canvas in the middle.

3. **Add & Duplicate Frames:** Use the '+' button in the timeline to add a new frame. Use the copy icon on a frame to duplicate it.

4. **Background:** Use the color picker for a solid color or 'Upload Image' for a picture background.

5. **Onion Skin:** Check this box to see faint outlines of your previous and next drawings.

6. **Play & Download:** Use the play button to preview your animation and the download button to save it as a `.webm` video file.
