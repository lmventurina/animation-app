How to Use the Animation Tool

1. Save/Load Project: Use the 'Save' and 'Load' buttons to save your progress as a `.anim` file and open it later.

2. Draw & Erase: Use the Pen and Eraser tools to draw on the canvas.

3. Lasso Delete: Use the Lasso tool to draw a selection around a large area. When you release the mouse, everything inside the selection will be deleted.

4. Add & Duplicate Frames: Use the '+' button in the timeline to add a new frame. Use the copy icon on a frame to duplicate it.

5. Background: Use the color picker for a solid color or 'Upload Image' for a picture background.

6. Onion Skin: Check this box to see faint outlines of your previous and next drawings.

7. Play & Download: Use the play button to preview your animation and the download button to save it as a `.webm` video file.
